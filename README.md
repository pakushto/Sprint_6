# Автотесты для сервиса «Самокат»

Набор UI‑тестов на Selenium + Pytest для учебного сервиса аренды самокатов (Практикум). Тесты оформлены по Page Object и снабжены шагами Allure.

## Состав репозитория
- `conftest.py` — фикстура `driver`, поднимает Firefox и открывает базовый URL.
- `data.py` — тестовые данные и конфиг (URL, тайм-ауты, входные данные для форм и FAQ).
- `locators/` — селекторы элементов главной страницы и страницы оформления заказа.
- `pages/` — Page Object: базовые обёртки ожиданий/кликов, страницы главная и заказа.
- `tests/` — сюда добавляются тесты (репозиторий содержит примерные артефакты Allure).
- `allure_results/` — сохранённые результаты прогона для генерации отчёта.
- `requirements.txt` — зависимости проекта.

## Требования
- Python 3.10+.
- Браузер Firefox и установленный в `PATH` geckodriver.
- (Опционально) Java + Allure CLI для генерации отчёта.

## Установка
```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск тестов
- Базовый прогон: `pytest -s -v tests`
- С сохранением шагов для отчёта Allure: `pytest -s -v --alluredir=allure_results tests`
- Просмотр отчёта: `allure serve allure_results`

## Настройки и доработки
- Базовые URL и тайм-аут находятся в `data.py` (`Urls`, `Config`).
- Для другого браузера измените фикстуру `driver` в `conftest.py`.
- При добавлении новых страниц и тестов придерживайтесь текущей структуры Page Object.
